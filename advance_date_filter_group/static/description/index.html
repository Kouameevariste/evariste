Advance Date Filters And Group By

For working on specific records, you can filter records with multiple date filters like Yearsterday , Today, Tomorrow, Current Week, Last Week, Last 7 Days, Current Month, Last Month, Last 30 Days, Current Year, Last Year, Last 365 Days and group by with Week, Month, Year


Key features:
  • Easy To Use.
  • Days Filter: Tomarrow, Toady, Yesterday, Last 7 Days, Last 30 Days, Last 365 Days.
  • Month Filter: Current Month, Last Month.
  • Year Filter: Current Year, Last Year.
  • Group By Week, Month, Year
  • To Many Filters? You Disable Days/Month/Year Filters From User Access Right Settings
  • Applied On Multiple Objects: Sale Orders, Claims, Logged Calls, Scheduled Calls, Purchase Orders, All Type Of Invoices, Task Stock Moves And All Transfer Transactions.
To enable date filters, GoTo Settings --> Users & Companies --> Users --> Assess Right --> Technical Settings.
